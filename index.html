🔖 项目简介

本程序为定制化请假条生成工具，通过智能图像处理技术快速生成符合学校规范的电子请假条。需配合APP请假界面的竖版截图使用。

📁 文件结构

核心文件要求：
  • input.jpg 必须存在且保持原始命名
  • 程序目录结构不可修改
.
├── MyApp.exe        # 主程序（需管理员权限运行）
├── input.jpg        # 空白假条模板（严禁删除）
├── output.jpg       # 生成的假条文件
└── config.ini       # 用户配置文件

🚀 使用流程

1. 准备素材

  • 获取APP竖版请假界面截图
  • 需包含完整系统状态栏
  • 保存到程序同目录

2. 配置编辑

  • 使用文本编辑器修改config.ini
  • 保留所有方括号标题
  • 所有字段必须完整填写

3. 运行程序

  • 双击运行MyApp.exe
  • 选择配置文件
  • 填写申请信息
  • 选择辅助照片

⚠️ 注意事项

截图规范

  • 必须包含完整系统状态栏
  • 推荐iOS/Android原生截图
  • 状态栏区域无遮挡

文件管理

  • 每次生成前自动清除旧文件
  • 定期备份config.ini
  • 禁止修改程序目录结构

系统参数

  • 向下压缩长度：默认40px
  • 取样长度：默认120px
  • 建议保持默认设置

🛠 技术支持

  • 运行环境：Windows 10/11 专业版
  • 需要管理员权限运行
  • 系统参数调整请联系开发者

  2. 配置config.ini文件
  3. 运行程序： 
    python main.py
  4. 填写表单并生成请假条
